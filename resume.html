    FM-Delta: Fault Management Packet Compression

    Cooperation with Marvell Technology | Oct 2014

    In this project we implemented a few packet compression/decompression algorithms for network devices. We compared the algorithms in terms of performance and compression ratio. The project included researching and analyzing well-known compression algorithms. We implemented a simple hardware-friendly delta encoding algorithm with of compression ratio of ~2.6. Working tools: Linux Ubuntu, VMWARE, C, Bash for scripting, Matlab for analyzing results and presenting graphs.
    We published the paper "FM-Delta Compressing Fault Management Packets" that was accepted by IEEE. The project poster participated at the IM 2017 conference in Lisbon, Portugal.

    Tunerino - Automatic Electric Guitar Tuner

    Final Project - Arduino and IoT | Feb 2016

    Tunerino - this product allows electric guitar players from all levels to tune their guitar quickly, friendly and without knowing the exact musical note. The player chooses a note with the application, press "Tune" and the tuner automatically tunes the string with a servo attached to the peg. The player may choose his favorite song or musical configuration (standard, drop C, etc) and the tuner will automatically tune is guitar accordingly. In this project we implemented a tuner circuit using Arduino, Windows Phone application and Microsoft Azure database.
